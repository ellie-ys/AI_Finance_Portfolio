# AI Finance Portfolio Overview

**Columbia Business School Executive Education: AI for Finance**

## 프로젝트 목표

- 금융 및 비즈니스 분야에서 AI 활용 실제 사례 학습
- 모듈별 실습 중심 학습 후 포트폴리오 구축
- 팀/조직에 AI 솔루션 적용 가능 수준 달성

## 학습 모듈 순서

1. AI 개론 (AI Overview)
2. 비즈니스를 위한 파이썬 (Business Python)
3. 예측 분석 (Predictive Analytics)
4. 생성형 AI (Generative AI)
5. 시뮬레이션 및 최적화 (Simulation & Optimization)
6. 실용 도구 활용 (Practical Tools)
7. 실무 사례 프로젝트 (01~07 각 폴더)

## 사용 도구

- Python, Pandas, Scikit-learn, Matplotlib
- OpenAI API, Microsoft Copilot
- 시뮬레이션/최적화 도구(PuLP, Monte Carlo 등)

## 포트폴리오 구조

- 각 폴더: 실무 사례 프로젝트 + 모듈 연결 + 코드/데이터/결과 포함
